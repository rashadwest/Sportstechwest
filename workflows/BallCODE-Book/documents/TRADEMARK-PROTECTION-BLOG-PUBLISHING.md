## üìã TRADEMARK BASICS

### What is a Trademark?

A trademark is a word, phrase, symbol, or design that identifies and distinguishes your goods/services from others.

**For BallCODE:**
- **Mark:** "BallCODE" (word mark)
- **Use:** Educational services, curriculum, software
- **Protection:** Prevents others from using confusingly similar marks

### Two Types of Trademark Rights

#### 1. **Common Law Trademark Rights** (Automatic)
- **When:** As soon as you use the mark in commerce
- **Protection:** Limited to geographic area where you use it
- **Symbol:** TM (‚Ñ¢)
- **Cost:** Free (automatic)
- **Strength:** Weaker protection

#### 2. **Federal Trademark Registration** (Filing Required)
- **When:** After filing with USPTO and approval
- **Protection:** Nationwide protection
- **Symbol:** ¬Æ (registered trademark)
- **Cost:** $250-350 per class (filing fees)
- **Strength:** Stronger protection, statutory damages

## üöÄ RECOMMENDED: FILE INTENT-TO-USE APPLICATION

### What is Intent-to-Use?

**Intent-to-Use (ITU) Application:**
- File before you actually use the mark commercially
- Reserves your right to the mark
- Must show actual use within 6 months (extendable to 3 years)
- Cost: $250-350 per class

### Why File ITU Now?

**Advantages:**
- ‚úÖ Establishes priority date (filing date)
- ‚úÖ Prevents others from registering similar marks
- ‚úÖ Shows serious intent to use
- ‚úÖ Can use TM symbol immediately
- ‚úÖ Can upgrade to ¬Æ after registration

**Timeline:**
- File ITU: Now (before blog publishes)
- Show actual use: Within 6 months (when you launch commercially)
- Registration: 8-12 months after filing

### What You Need to File ITU:

1. **Mark:** "BallCODE"
2. **Goods/Services:** Educational services, curriculum, software
3. **Basis:** Intent to use
4. **Owner:** Rashad West (or your entity)
5. **Filing Fee:** $250-350 per class

**Classes to Consider:**
- **Class 41:** Educational services, curriculum
- **Class 42:** Software, computer services
- **Class 9:** Software, downloadable software

### What Happens If You Don't Register?

**Risks:**
- Limited geographic protection
- Weaker position in disputes
- No statutory damages
- Harder to enforce rights

- Still have common law rights
- Can still use TM symbol

**Risks:**
- They get priority
- You may be forced to rebrand
- Costly legal disputes
- Lost investment in brand

**Prevention:**
- File ITU application now
- Establish priority date
- Protect your investment
