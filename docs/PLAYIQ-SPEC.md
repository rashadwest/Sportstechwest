# PlayIQ — Spec (Offseason, Friendlies, Pickup Runs)

**Working name:** PlayIQ  
**Tagline (draft):** *Move like it’s practice — because it is.*

---

## Positioning

PlayIQ is **play-first movement** for low-stakes days: **offseason runs, friendlies, scrimmages, pickup games, and “just get moving” sessions.** Not peak performance or grind — **joy, variety, and showing up.**

| | WorkoutIQ / similar | **PlayIQ** |
|--------|---------------------|------------|
| Tone | Intensity, structure | Light, social, optional |
| Stakes | Training blocks | Fun, exploration, connection |
| Best for | In-season, programs | Offseason, friendlies, weekends |

---

## Audience

- Athletes in **offseason** or **deload** weeks  
- **Rec leagues**, **pickup**, **friendly matches**  
- **Group runs** where the point is the group, not the PR  
- Anyone who wants **movement without a spreadsheet**

---

## Formats (same IQ system as mentalIQ)

| Format | Role |
|--------|------|
| **~3 min** | Hook + one “play rule” (e.g. “every chorus = switch direction”) — shareable, TikTok/Reels |
| **~1 hr** | Long session: rotating games, easy prompts, music-friendly pacing, **5 sec between segments** |

**Audio:** Voice-led; optional music bed. **Video:** Brain loop + assemble (same pipeline as mentalIQ).

---

## Content pillars (for scripts)

1. **Rules light enough to remember** — one clear game per block (e.g. “shadow the person in front,” “add one silly move each lap”).  
2. **Social hooks** — “bring a friend,” “first one to laugh buys water” — optional, not required.  
3. **Offseason honesty** — permission to go easy, try something new, or skip metrics.  
4. **Friendly / pickup framing** — “competition is the game, not the opponent’s soul.”

---

## Differentiation from mentalIQ

- **mentalIQ** = mind: breath, focus, reset.  
- **PlayIQ** = body + context: **fun, variety, low-stakes** movement.  
They can be **paired**: 3 min mentalIQ meditation → 3 min PlayIQ intro → longer run.
