## üìö CURRICULUM STRUCTURE: GRADES 6-8

### Grade 6: Foundation Building (Block Coding ‚Üí Python Introduction)
**Focus:** Introduction to computer science concepts through visual blocks, transitioning to Python basics

**CSTA Standards Addressed:**
- **1B-AP-10:** Create programs that include sequences, events, loops, and conditionals
- **1B-AP-11:** Decompose problems into smaller, manageable parts
- **1B-AP-12:** Modify, remix, or incorporate portions of existing programs
- **1B-AP-15:** Test and debug programs

**Block Coding Level (Phase 1):**
- **Scratch-Level Complexity:** Grades 3-5 equivalent (accessible entry point)
- **Concepts:** Sequences, conditionals, loops
- **Assessment:** 60% block coding, 20% code reading, 20% code writing

**Python Introduction (Phase 3):**
- **Basic Python:** `print()`, variables, simple sequences
- **Code Reading:** Understand Python syntax
- **Code Writing:** Write simple Python programs

**AIMCODE Expert Alignment:**
- **Resnick (Constructionist):** Students build programs with blocks
- **Reggio (Multiple Entry Points):** Story, blocks, code, game
- **Hassabis (Systematic):** Foundation layer before complexity

---

### Grade 7: Skill Development (Python Programming)
**Focus:** Intermediate Python programming with data analysis

**CSTA Standards Addressed:**
- **2-AP-11:** Create clearly named variables that represent different data types
- **2-AP-12:** Design and iteratively develop programs that combine control structures
- **2-AP-13:** Decompose problems and subproblems into parts
- **2-AP-16:** Incorporate existing code, media, and libraries into original programs

**Python Level (Phase 3):**
- **Intermediate Python:** Conditionals, loops, functions
- **Data Structures:** Lists, dictionaries
- **Real Data:** Work with basketball statistics

**AIMCODE Expert Alignment:**
- **Zhang (Story Context):** Concepts emerge from basketball narrative
- **Resnick (Building):** Students create programs, not just consume
- **Hassabis (Layer 2):** Building on foundation, adding complexity

---

### Grade 8: Mastery & Career Preparation (Advanced Python)
**Focus:** Advanced Python programming with analytics integration

**CSTA Standards Addressed:**
- **3A-AP-13:** Create prototypes that use algorithms to solve computational problems
- **3A-AP-16:** Design and iteratively develop computational artifacts
- **3A-AP-17:** Decompose problems into smaller components through systematic analysis
- **3A-AP-18:** Create artifacts by using procedures within a program

**Python Level (Phase 3):**
- **Advanced Python:** Functions, modules, algorithms
- **Data Analysis:** Statistics, visualization
- **Real Platform:** BallCODE Jr. Analytics (BTE Analytics integration)

**AIMCODE Expert Alignment:**
- **Hassabis (Layer 3):** Advanced concepts, systems thinking
- **Jobs (Simple Design):** Complex concepts made accessible
- **Zhang (Real-World):** Career pathway integration

## üîÑ THREE-PHASE LEARNING PATHWAY (AIMCODE-Aligned)

### Phase 1: Sports Language (Block Coding) - Grades 6-8
**AIMCODE Alignment:** Resnick (Constructionist) + Reggio (Multiple Entry Points)

**What Students Learn:**
- **Grade 6:** Sequences, basic conditionals, simple loops (Scratch-level)
- **Grade 7:** Complex conditionals, nested loops, variables (Intermediate Scratch)
- **Grade 8:** Functions, arrays, algorithms (Advanced Scratch ‚Üí Python bridge)

**CSTA Standards:**
- **1B-AP-10:** Create programs with sequences, events, loops, conditionals
- **1B-AP-11:** Decompose problems into smaller parts
- **1B-AP-12:** Modify and remix programs

**Scratch/Block Coding Alignment:**
- **Grades 3-5 Equivalent:** Basic block coding (accessible entry point)
- **Grades 6-8 Standard:** Complex block structures, variables, functions
- **Justification:** Students learn CS concepts before syntax complexity

**Why This Works:**
- Visual learning reduces cognitive load
- Basketball context makes concepts accessible
- Multiple entry points (story, blocks, game)
- Builds confidence before text-based coding

---

### Phase 2: Transition Bridge - Grades 6-8

**CSTA Standards:**

- Systematic progression (Hassabis)

**AIMCODE Alignment:** Zhang (Story Context) + Resnick (Building)

**CSTA Standards:**

- **3A-AP-16:** Design computational artifacts

**Python Grade Level Alignment:**
- **Grade 6:** Introduction to Python (industry-standard language)
- **Grade 7:** Intermediate Python (real programming skills)
- **Grade 8:** Advanced Python (career-ready skills)

**Justification:**
- Students learn real programming language (Python)
