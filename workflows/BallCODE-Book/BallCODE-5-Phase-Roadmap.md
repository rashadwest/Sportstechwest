# BallCODE MVP & Sales Execution - 5 Phase Roadmap

## Overview
5-phase execution plan to complete Episode 1 MVP and launch educational partnership outreach.

**Start Date:** Monday (Week 1, Day 1)  
**Total Duration:** 6 weeks (phases may overlap)

---

## Phase 1: Foundation & Content Creation (Weeks 1-2)

### Goals
- Set up project tracking and organization systems
- Complete Episode 1 story content (all 3 acts)
- Create Skill Pit-Stop content
- Design all exercises
- Launch initial outreach campaign

### Week 1: Setup & Story Start

**Monday - Project Setup**
- Set up Notion database for Episode 1 tracking
- Create Episode 1 project page with all sections
- Review existing Episode 1 outline
- Set up daily tracking system
- Research 10-15 target schools/districts for pilot programs
- Create outreach spreadsheet
- Draft initial outreach email template

**Tuesday - Story Content Start**
- Write Act I of Episode 1 story (400-600 words)
- Research local/regional educational conferences, STEM events
- Add 5-10 conference/event opportunities to outreach spreadsheet
- Identify 3-5 potential partnership organizations

**Wednesday - Story Content Continue**
- Write Act II of Episode 1 story (400-600 words)
- Draft pilot program proposal document (1-2 pages)

**Thursday - Story Content Complete**
- Write Act III of Episode 1 story (400-600 words)
- Complete full story draft (1,200-1,800 words total)
- Refine outreach email template based on pilot proposal
- Create email variations

**Friday - Story Review & Outreach Launch**
- Self-edit Episode 1 story draft
- Check alignment with objectives and concepts
- Send first batch of 5 outreach emails to target schools
- Set calendar reminders for follow-ups

### Week 2: Skill Content & Visual Planning

**Monday - Skill Pit-Stop Content**
- Write Skill Pit-Stop mini-lesson (300-500 words)
- Follow up on Week 1 outreach emails
- Send second batch of 5 outreach emails

**Tuesday - Exercise Design**
- Design Exercise A: "Label states in a play"
- Design Exercise B: "Write state transitions"
- Design Exercise C: "Handle an edge case"
- Create answer keys/teacher guides
- Research visual creation tools/processes
- Create detailed visual briefs for Episode 1 assets

**Wednesday - Game Integration Planning**
- Review Unity scripts
- Map Episode 1 exercises to Training Mode
- Design QR code/short URL structure
- Create integration spec
- Continue outreach: Send 3-5 more emails
- Respond to inquiries and schedule discovery calls

**Thursday - Visual Assets Creation Start**
- Create or commission Episode 1 visual assets
- Outreach follow-ups and new contacts
- Prepare for scheduled discovery calls

**Friday - Content Integration**
- Integrate story, Skill Pit-Stop, and exercises into single document
- Format for picture book layout
- Add placeholder spots for visuals
- Create PDF draft version
- Conduct discovery calls (if scheduled)
- Plan Week 3 outreach strategy

**Phase 1 Deliverables:**
- [ ] Episode 1 story complete (all 3 acts)
- [ ] Skill Pit-Stop content written
- [ ] All 3 exercises designed
- [ ] Visual briefs created
- [ ] 15-20 schools contacted
- [ ] Outreach tracking system operational

## Phase Overlap & Parallel Work

**Note:** Phases are designed to allow parallel work:
- Outreach continues throughout all phases
- Content creation can overlap with technical development
- Beta testing can begin while finalizing visuals
- Pilot programs can launch while iterating on content

**Daily Time Commitment:**
- Product Development: 4-6 hours/day
- Sales/Outreach: 2-3 hours/day
- Total: 6-9 hours/day

---

## Risk Mitigation

**If Episode 1 takes longer:**
- Prioritize story + Skill Pit-Stop + 1 exercise (minimum viable)
- Defer some visual polish to Phase 5
- Game integration can be simplified initially

**If outreach response is low:**
- Expand target list (after-school programs, coding camps, homeschool groups)
- Offer more flexible pilot terms
- Create free sample content to build interest
- Leverage personal networks and referrals

**If pilot programs need more content:**
- Use Episode 1 as proof of concept
- Offer to create Episode 2 during pilot period
- Provide additional exercises/worksheets as needed
