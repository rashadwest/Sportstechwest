## üé® DESIGN SYSTEM SPECIFICATIONS

### Colors (Kid-Friendly Palette):
```css
/* Primary Colors */
--primary-orange: #FF6B35;      /* Energetic, fun */
--primary-blue: #4ECDC4;         /* Calm, trustworthy */
--primary-green: #95E1D3;       /* Success, growth */

/* Accent Colors */
--accent-yellow: #FFE66D;       /* Joy, energy */
--accent-pink: #FF8B94;         /* Playful, fun */
--accent-purple: #A8DADC;       /* Creative, calm */

/* Neutral Colors */
--neutral-dark: #2C3E50;        /* Text, headings */
--neutral-light: #ECF0F1;       /* Backgrounds */
--neutral-white: #FFFFFF;       /* Cards, surfaces */

/* Status Colors */
--success: #2ECC71;              /* Green - success */
--warning: #F39C12;              /* Orange - warning */
--error: #E74C3C;                /* Red - error */
```

### Typography:
```css
/* Headings - Playful but readable */
--font-heading: 'Comic Sans MS', 'Nunito', sans-serif;
--font-size-h1: 2.5rem;         /* 40px */
--font-size-h2: 2rem;           /* 32px */
--font-size-h3: 1.5rem;         /* 24px */

/* Body - Clear and readable */
--font-body: 'Nunito', 'Open Sans', sans-serif;
--font-size-body: 1rem;         /* 16px */
--font-size-small: 0.875rem;    /* 14px */

/* Buttons - Bold and clear */
--font-button: 'Nunito', sans-serif;
--font-size-button: 1.125rem;   /* 18px */
--font-weight-button: 700;     /* Bold */
```

### Spacing (8px Grid System):
```css
--spacing-xs: 4px;    /* 0.5 * 8 */
--spacing-sm: 8px;    /* 1 * 8 */
--spacing-md: 16px;   /* 2 * 8 */
--spacing-lg: 24px;   /* 3 * 8 */
--spacing-xl: 32px;   /* 4 * 8 */
--spacing-xxl: 48px;  /* 6 * 8 */
```

### Buttons:
```css
/* Primary Button - Big, colorful, obvious */
.button-primary {
  background: var(--primary-orange);
  color: white;
  padding: 16px 32px;
  border-radius: 12px;
  font-size: 1.125rem;
  font-weight: 700;
  border: none;
  cursor: pointer;
  transition: all 0.3s ease;
  box-shadow: 0 4px 12px rgba(255, 107, 53, 0.3);
}

.button-primary:hover {
  transform: translateY(-2px);
  box-shadow: 0 6px 16px rgba(255, 107, 53, 0.4);
}

.button-primary:active {
  transform: translateY(0);
}

/* Secondary Button */
.button-secondary {
  background: var(--primary-blue);
  color: white;
  /* Same structure as primary */
}

/* Success Button (for celebrations) */
.button-success {
  background: var(--success);
  color: white;
  /* Same structure */
}
```

### Cards (Notion-Inspired):
```css
.card {
  background: white;
  border-radius: 12px;
  padding: 24px;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
  transition: all 0.3s ease;
}

.card:hover {
  transform: translateY(-4px);
  box-shadow: 0 4px 16px rgba(0, 0, 0, 0.15);
}
```

## üöÄ IMPLEMENTATION ORDER

### ‚ö†Ô∏è CRITICAL: LOCALHOST FIRST
**All UI/UX work must be done on localhost for preview before pushing!**

**Workflow:**
1. **Make changes locally** (in `BallCode/` directory)
2. **Test on localhost** (`http://localhost:8080` or similar)
3. **Preview and review** with user
4. **Get approval** before pushing
5. **Then push** to repository/Netlify

**Local Development Setup:**
```bash
# Navigate to BallCode directory
cd BallCode

# Use local server (Python, Node, or Netlify CLI)
# Option 1: Python
python -m http.server 8080

# Option 2: Node (if you have http-server)
npx http-server -p 8080

# Option 3: Netlify CLI (for exact Netlify preview)
netlify dev
```

**Preview URL:** `http://localhost:8080` or `http://localhost:8888` (Netlify)
