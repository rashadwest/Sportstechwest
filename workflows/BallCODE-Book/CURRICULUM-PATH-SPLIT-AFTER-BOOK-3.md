## üéØ BOOK 3: THE BRIDGE

**Book 3 serves as the bridge between block coding and Python:**

### What Book 3 Does:
1. **Primary Method:** Block coding (students practice with blocks)
2. **Python Preview:** Shows Python syntax as optional preview
3. **Concept Introduction:** Introduces idea that coding can be text-based
4. **Path Preparation:** Prepares students to choose their path

### How Book 3 Presents Loops:

**Block Coding (Primary):**
```
REPEAT [fake left] 3 TIMES
THEN [go right]
```

**Python Preview (Optional):**
```python
# This is Python code (optional preview)
for i in range(3):
    fake_left()
go_right()
```

**Message to Students:**
- "You can use blocks (easier) or Python (more advanced)"
- "Both do the same thing - repeat patterns"
- "After Book 3, you can choose your path"
