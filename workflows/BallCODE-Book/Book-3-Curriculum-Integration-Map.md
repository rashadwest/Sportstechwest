## üìö THREE-PHASE LEARNING PATHWAY

### Phase 1: Sports Language (Block Coding)

**What Students Do:**
- Learn loops using basketball terminology
- Visual block interface: REPEAT [fake left] 3 TIMES, THEN [go right]
- Drag blocks to create loop patterns
- Immediate feedback through game exercises

**Block Coding Example:**
```
REPEAT [Block 3: Fake Left] 3 TIMES
THEN [Block 3: Go Right]
```

**Learning Objective:**
- Students understand that loops repeat patterns
- Students can create loop patterns using blocks
- Students see that breaking loops creates opportunity

**Assessment:**
- Block coding exercises in game
- Pattern creation challenges
- Loop completion tasks

---

### Phase 2: Transition Bridge

**What Students Do:**
- See how loop blocks map to Python code
- Side-by-side view: Block loop | Python `for` loop
- Understand that blocks = code
- See Python syntax alongside blocks

**Bridge Example:**
```
Block: REPEAT [fake left] 3 TIMES
Code:  for i in range(3):
           fake_left()
```

**Learning Objective:**
- Students understand that loop blocks = Python `for` loops
- Students see Python loop syntax in context
- Students recognize loop patterns in code

**Assessment:**
- Side-by-side comparison exercises
- Pattern recognition activities
- Code reading (not writing yet)

---

### Phase 3: Python Learning

**What Students Do:**
- Write Python `for` loops
- Use `range()` to control repetitions
- Create loop patterns in code
- Break loops at the right moment

**Python Code Example:**
```python
# Create pattern: repeat 3 times
for i in range(3):
    fake_left()
    print(f"Fake {i+1}")

# Break pattern: go right
go_right()
print("Pattern broken!")
```

**Learning Objective:**
- Students can write Python `for` loops
- Students understand `range()` function
- Students apply loops to create patterns
- Students are ready for advanced loops

**Assessment:**
- Python code writing exercises
- Loop pattern projects
- Real-world problem solving
