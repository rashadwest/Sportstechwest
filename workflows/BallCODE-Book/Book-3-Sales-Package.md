## üìö CURRICULUM ALIGNMENT

### Learning Objectives

**Primary Objective:**
Students will understand and apply Python loops (`for` loops, `range()`) through basketball pattern creation and deception.

**Specific Skills:**
1. **Pattern Recognition:** Identify repeating patterns in basketball moves
2. **Loop Creation:** Create loops using visual blocks
3. **Python Syntax:** Write `for` loops with `range()` in Python
4. **Pattern Breaking:** Understand when to break loops for maximum effect
5. **Code Application:** Apply loops to solve real problems

### Standards Alignment

**CSTA Standards:**
- 1B-AP-12: Create programs using loops
- 2-AP-13: Decompose problems into smaller parts
- 2-AP-17: Test and debug programs

**Common Core Mathematics:**
- Pattern recognition and sequences
- Logical reasoning
- Problem-solving strategies

**NGSS:**
- Science and Engineering Practices
- Computational thinking
- Systems thinking

### Grade-Level Progression

| Grade Level | Focus | Python Skills |
