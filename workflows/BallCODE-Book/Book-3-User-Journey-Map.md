### Step 1: Discovery

**What They See:**

**User Action:** Click on Book 3 card

**Success Criteria:** Student understands what Book 3 teaches

---

### Step 2: Book Page

**Entry Point:** `ballcode.co/books/book3`

**What They See:**
1. **Hero Section:**
   - Book cover image (white character)
   - Title: "The Pattern (In & Out Dribble)"
   - Subtitle: "Learning Loops Through Basketball Deception"

2. **Story Section:**
   - Video player (when available)
   - Story text (when written)
   - Python code examples integrated
   - Visual: White character in basketball scenes

3. **Curriculum Connection:**
   - Clear learning pathway display
   - Phase 1 (Blocks) ‚Üí Phase 2 (Bridge) ‚Üí Phase 3 (Python)
   - Grade level indicators (3-5, 6-8, 9-12)

4. **Game Exercise:**
   - "Try the Exercise" button
   - Description: "Practice loops in the Deception Loop Challenge"
   - Visual preview of exercise

5. **Python Progression:**
   - Block code example
   - Python code example
   - Side-by-side comparison

6. **What's Included:**
   - Interactive video book
   - Game exercise access
   - Python practice materials
   - Curriculum alignment

**User Action:** Read/watch story, then click "Try the Exercise"

**Success Criteria:** Student understands loops concept and wants to practice

---

### Step 3: Game Exercise

**Entry Point:** `ballcode.co/play?book=3&exercise=deception-loop&source=book`

**What They Experience:**
1. **Unity Game Loads:**
   - Mathlete Mode activates
   - Exercise: "Deception Loop Challenge"
   - Instructions: "Create a loop pattern to get past the defender"

2. **Block Coding Interface:**
   - Drag blocks to create loop
   - LOOP [BLOCK_3_FAKE_LEFT] REPEAT 3, THEN [BLOCK_3_GO_RIGHT]
   - Visual feedback: Pattern building (1, 2, 3...)

3. **Execution:**
   - Click "Run" to execute
   - White character performs pattern
   - Defender reacts to pattern
   - Pattern breaks at right moment
   - Success: Get past defender

4. **Feedback:**
   - Success message
   - Score displayed
   - Python code shown (side-by-side with blocks)

**User Action:** Complete exercise, see success

**Success Criteria:** Student successfully creates loop pattern and gets past defender

---

### Step 4: Return to Book

**Entry Point:** Return from game to `ballcode.co/books/book3?exercise=complete&success=1&score=100`

**What They See:**
1. **Success Message:**
   - "Exercise Complete! üéâ"
   - Score displayed
   - Achievement unlocked

2. **Python Practice Unlocked:**
   - "View Python Code" button
   - Side-by-side: Block code | Python code
   - Interactive code examples

3. **Next Steps:**
   - "Continue to Python Practice"
   - "Try Advanced Patterns"
   - "Move to Next Book"

**User Action:** View Python code, continue learning

**Success Criteria:** Student sees Python connection and wants to learn more

---

### Step 5: Python Practice

**Entry Point:** Python practice section on Book 3 page

**What They Do:**
1. **See Code Examples:**
   - Block code they created
   - Python code equivalent
   - Side-by-side comparison

2. **Write Python Code:**
   - Copy/paste examples
   - Modify code
   - Run code (if code runner available)

3. **Understand Connection:**
   - "This block = this Python code"
   - "Loops repeat patterns in code"
   - "I can code!"

**User Action:** Write Python code, understand connection

**Success Criteria:** Student can write basic Python loops

---

### Step 6: Mastery & Progression

**Entry Point:** Mastery achieved

**What They See:**
1. **Mastery Badge:**
   - "Loop Master" achievement
   - Progress tracked
   - Next book unlocked

2. **Advanced Options:**
   - "Want to Go Deeper?" section
   - Link to "Learn to Code with Basketball"
   - Real data exercises

3. **Next Book:**
   - Book 4 preview
   - "Continue Learning" button
   - Progress pathway visible

**User Action:** Continue to next book or explore advanced options

**Success Criteria:** Student feels accomplished and wants to continue
