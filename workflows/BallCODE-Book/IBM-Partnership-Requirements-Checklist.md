#### Student Progress Tracking
- [ ] **Episode Completion Tracking**

  - Track partial completions (where students stop)
  - Store completion percentage per episode

- [ ] **Exercise Attempt Data**
  - Track every exercise attempt (success/failure)
  - Record number of attempts before success

  - Store code submissions (block coding and Python)

- [ ] **Skill Mastery Tracking**

- [ ] **Engagement Metrics**
