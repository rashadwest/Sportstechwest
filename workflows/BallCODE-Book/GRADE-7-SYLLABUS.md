## ðŸ“‹ COURSE DESCRIPTION

BallCODE Grade 7 builds on Grade 6 foundations, advancing students to intermediate Python programming and introducing data analysis using real basketball statistics. Students review and extend sequences, conditionals, and loops while learning to analyze data, create visualizations, and solve real-world problems. This course prepares students for Grade 8's BallCODE Jr. Analytics program while meeting all 7th grade educational standards.

**Core Philosophy:** 70% story, 30% skill instruction - now with real data analysis and professional tools.
