# IBM Partnership: Crawl, Walk, Run

**Two Platforms, One Partnership Strategy**

---

## The Two Platforms

### 1. **BTE Analytics** - Data & Statistics Platform
**What it is:** Basketball statistics, data, and analytics platform  
**What IBM gets:** Real basketball data, statistics, analytics insights  
**Partnership type:** Data partner - IBM gets access to statistics, data, and analytics

### 2. **BallCODE** - Storytelling Game Platform
**What it is:** Educational game that teaches kids coding/math/AI through basketball stories  
**What IBM gets:** Platform to work with kids, educational content, learning data  
**Partnership type:** Gaming partner - IBM uses the game to work with kids, long-term data integration

### How They Work Together
- **BTE Analytics data** makes **BallCODE game** more realistic
- **BallCODE game** generates learning analytics that can feed back into insights
- Together, they create a complete ecosystem: real data + engaging education
