### **Step 2: Put It in GitHub (Like Putting a Password in a Safe)**

**This is like putting a password in a safe so the robot can use it:**

1. **Go to this website:**
   ```
   https://github.com/rashadwest/BTEBallCODE/settings/secrets/actions
   ```
   (I'll open this for you)

2. **Click the green button that says "New repository secret"**

3. **Fill in the form:**
   - **Name:** Type: `UNITY_SERIAL` (if you got a number)
     - OR Type: `UNITY_LICENSE` (if you got a file)
   - **Secret:** Paste the number or file contents here

4. **Click "Add secret"**

5. **Done!** The robot now has the password!
