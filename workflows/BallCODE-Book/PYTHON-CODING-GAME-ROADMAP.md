## üéÆ PYTHON GAME MODE CONCEPT

### ELI10: What Is Python Coding Game?

**Think of it like writing a recipe, but in Python code instead of blocks!**

**How It Works:**
1. **You see a code editor** (like a text box)
2. **You write Python code** using basketball words:
   ```python
   start()
   pound_dribble()
   pound_dribble()
   pound_dribble()
   advance()
   bucket()
   ```

3. **You click "RUN"** and watch your robot do what you coded!

4. **You see your code work** just like the blocks did!

**The Connection:**
- **Blocks:** START ‚Üí POUND ‚Üí POUND ‚Üí POUND ‚Üí BUCKET
- **Python:** `start()`, `pound_dribble()`, `pound_dribble()`, `pound_dribble()`, `bucket()`
- **Same thing, different language!**

**What You're Learning:**
- How to write Python code
- How blocks = Python code
- Real programming skills
- Basketball terminology in code

**Goal:** Test the concept, see if students like it

**Level 1: "Your First Python Code"**
- **Concept:** Write simple sequence in Python
- **Task:** Write code to pound 3 times, then shoot
- **Python:**
  ```python
  start()
  pound_dribble()
  pound_dribble()
  pound_dribble()
  bucket()
  ```
- **Learning:** Basic Python syntax, sequences
- **Difficulty:** Beginner

**Level 2: "Blocks to Python"**
- **Concept:** See blocks, write Python equivalent
- **Task:** Given blocks, write Python code
- **Blocks:** START ‚Üí CROSSOVER ‚Üí IN & OUT ‚Üí BUCKET
- **Python:**
  ```python
  start()
  crossover_dribble()
  in_out_dribble()
  bucket()
  ```
- **Learning:** Translation from blocks to code
- **Difficulty:** Beginner
