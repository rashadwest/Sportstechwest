## üéÆ BLOCK CODING LEVEL IDEATIONS

### Foundation Levels (Book 1 - Sequences):

#### Level 1.1: "First Pound"
**Concept:** Simple sequence with one move
- **Blocks:** START, DRIBBLE 1 (Pound), BUCKET
- **Goal:** Pound 3 times, then shoot
- **Learning:** Basic sequence, one move repeated
- **Difficulty:** Beginner

#### Level 1.2: "Pound Power"
**Concept:** Multiple pounds in sequence
- **Blocks:** START, DRIBBLE 1 (Pound √ó 5), BUCKET
- **Goal:** Pound 5 times to break through defense
- **Learning:** Repetition, counting
- **Difficulty:** Beginner

#### Level 1.3: "Pound and Advance"
**Concept:** Pound + movement
- **Blocks:** START, DRIBBLE 1 (Pound), ADVANCE, BUCKET
- **Goal:** Pound, move forward, shoot
- **Learning:** Multiple different actions
- **Difficulty:** Beginner

### Progression Levels (Book 2 - Conditionals):

#### Level 2.1: "Crossover Choice"
**Concept:** First conditional (if/then)
- **Blocks:** START, IF [defender close] THEN [Crossover] ELSE [Advance], BUCKET
- **Goal:** Read defense, make decision
- **Learning:** Conditionals, decision-making
- **Difficulty:** Intermediate

#### Level 2.2: "Defender Read"
**Concept:** Multiple conditionals
- **Blocks:** START, IF [defender left] THEN [go right] ELSE IF [defender right] THEN [go left] ELSE [go straight], BUCKET
- **Goal:** Read multiple defenders
- **Learning:** Complex conditionals
- **Difficulty:** Intermediate

### Advanced Levels (Book 3 - Loops):

#### Level 3.1: "Pattern Practice"
**Concept:** Simple loop
- **Blocks:** START, REPEAT 3 TIMES [Pound], BUCKET
- **Goal:** Create pattern with loop
- **Learning:** Loops, repetition
- **Difficulty:** Intermediate

#### Level 3.2: "Deception Loop"
**Concept:** Loop with conditionals
- **Blocks:** START, REPEAT 3 TIMES [In & Out], THEN [Crossover], BUCKET
- **Goal:** Create pattern, break it at right time
- **Learning:** Loops + conditionals
- **Difficulty:** Advanced

### Advanced Sequences:

#### Level 4.1: "All 7 Dribbles"
**Concept:** Use all dribbles in sequence
- **Blocks:** START, DRIBBLE 1, 2, 3, 4, 5, 6, 7, BUCKET
- **Goal:** Complete full sequence
- **Learning:** All moves, complex sequences
- **Difficulty:** Advanced
