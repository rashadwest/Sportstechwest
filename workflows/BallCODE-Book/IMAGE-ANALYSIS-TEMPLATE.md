# Quick Image Analysis Template

**For each image, provide:**
1. Level name/number
2. Objective (what player needs to do)
3. Court layout (player position, basket, obstacles)
4. Blocks available in palette
5. Blocks arranged in solution
6. Coding concept being taught
7. Game state (running/paused/success/failure)

**Copy this format for each image:**

```
Image: [filename]
Level: [name/number]
Objective: [what to do]
Court: [describe layout]
Blocks Available: [list them]
Blocks Arranged: [what's shown]
Concept: [what's being taught]
State: [running/paused/etc]
```
