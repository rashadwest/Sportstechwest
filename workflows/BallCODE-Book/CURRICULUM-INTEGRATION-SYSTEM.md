## üîÑ THE COMPLETE FLOW

### Phase 1: Website (Entry Point)
**What Students See:**
- Clean, organized website
- Books section with curriculum pathway visible
- Each book shows:
  - What Python concept is taught
  - What basketball skill is learned
  - What grade levels it's for
  - What curriculum standards it aligns with
  - What game exercise connects to it

**Integration Points:**
- Curriculum pathway visible on homepage
- Book cards show learning objectives
- Clear progression indicators

---

### Phase 2: Book (Learning Content)
**What Students Experience:**
- Read/watch the book (story + video)
- Learn Python concept through basketball story
- See curriculum connection clearly marked
- Understand what they're learning and why

**Integration Points:**
- **"What You're Learning" Section:**
  - Python concept (e.g., "Loops and Repetition")
  - Basketball skill (e.g., "In & Out Dribble")
  - Grade level (e.g., "Grades 3-12")
  - Curriculum standards (CSTA, Common Core, NGSS)

- **"Try the Exercise" Button:**
  - Links to game exercise
  - Shows what they'll practice
  - Clear connection to book content

---

### Phase 3: Game (Hands-On Practice)
**What Students Do:**
- Complete game exercise
- Practice Python concept through basketball
- Get immediate feedback
- See their code in action

**Integration Points:**
- Exercise matches book content
- Same Python concept
- Same basketball skill
- Progress tracked and returned to website

---

### Phase 4: Curriculum (Learning Guide)
**What Students See:**
- "What You Learned" summary
- Python code examples
- Curriculum pathway showing:
  - Where they are now
  - What they've completed
  - What's next
  - How concepts build on each other

**Integration Points:**
- **"Next Book" Recommendation:**
  - Based on curriculum progression
  - Shows logical next step
  - Explains why this book comes next
  - Clear learning pathway
