# IBM SkillsBuild Teacher Toolkit Integration Plan
## Incorporating Career Readiness into BallCODE Curriculum

**Date:** 2025  
**Purpose:** Integrate IBM SkillsBuild career readiness components into BallCODE's coding-through-basketball curriculum  
**Target:** Enhance curriculum with career exploration, resume writing, and interview preparation

---

## Executive Summary

The IBM SkillsBuild Teacher Toolkit provides valuable career readiness resources that complement BallCODE's technical curriculum. This integration plan outlines how to incorporate career exploration, resume writing, and interview preparation into the existing 12-episode structure while maintaining the core sports-to-code learning pathway.

**Key Integration Points:**
- Career connections for each coding concept
- Career exploration activities as extension exercises
- Resume building using coding projects as experience
- Interview preparation with technical focus
- Career pathway mapping from coding skills to jobs

## Integration Strategy

### Phase 1: Career Connections (Episodes 1-12)
**Add career relevance to each existing episode**

#### Implementation Approach
- **Career Spotlight Section** in each episode
- **Career Connection Activity** in teacher guides
- **Extension Exercise** linking coding concept to real-world careers

#### Example: Episode 1 Integration
**Career Spotlight:**
- **State Management** → Software Engineer, Game Developer, Systems Analyst
- **Real-World Application:** Video games, mobile apps, web applications all use state management
- **Career Path:** Start with coding → Junior Developer → Senior Developer → Tech Lead

**Career Connection Activity:**
- Students research one career that uses state management
- Create a "Day in the Life" description
- Identify 3 skills needed for that career
- Connect those skills to what they learned in Episode 1

**Extension Exercise:**
- "Career Interview": Students interview a software developer (or watch video interview)
- Identify where state management is used in their work
- Write a reflection connecting Episode 1 concepts to the career

---

### Phase 2: Career Readiness Module (Post-Episode 12)
**Standalone module for career preparation**

#### Module Structure
1. **Career Exploration Workshop** (2-3 class periods)
   - Explore tech careers related to coding
   - Identify interests and strengths
   - Map coding skills to career paths

2. **Resume Building Workshop** (2-3 class periods)
   - Use BallCODE projects as experience
   - Highlight coding skills learned
   - Create technical resume format

3. **Interview Preparation Workshop** (2-3 class periods)
   - Technical interview basics
   - Practice explaining coding concepts
   - Mock interviews with coding questions

#### Module Components

**Career Exploration Workshop:**
- **Activity 1:** Career Interest Assessment
  - Students identify which coding concepts they enjoyed most
  - Match interests to career paths
  - Research 3 careers in tech

- **Activity 2:** Career Pathway Mapping
  - Map BallCODE episodes to career skills
  - Identify education requirements
  - Create personal career pathway

- **Activity 3:** Industry Guest Speaker (or video)
  - Software developer, data scientist, game developer
  - Q&A about career and coding
  - Connect to BallCODE concepts

**Resume Building Workshop:**
- **Activity 1:** Skills Inventory
  - List all coding concepts learned (Episodes 1-12)
  - Identify projects completed
  - Document achievements

- **Activity 2:** Resume Template Creation
  - Use IBM SkillsBuild resume format
  - Add BallCODE projects as experience
  - Highlight technical skills section

- **Activity 3:** Portfolio Development
  - Create portfolio of BallCODE projects
  - Write project descriptions
  - Link to GitHub or portfolio site

**Interview Preparation Workshop:**
- **Activity 1:** Technical Interview Basics
  - Types of technical interviews
  - Common coding questions
  - How to explain coding concepts

- **Activity 2:** Practice Interviews
  - Pair students for mock interviews
  - Practice explaining Episode concepts
  - Give and receive feedback

- **Activity 3:** Behavioral Interview Prep
  - STAR method (Situation, Task, Action, Result)
  - Practice with BallCODE project examples
  - Connect projects to teamwork, problem-solving

### Phase 3: Enhanced Teacher Guides
**Add career components to existing teacher resources**

#### Teacher Guide Additions

**For Each Episode, Add:**

1. **Career Connections Section**
   - 3-5 careers that use this coding concept
   - Real-world examples
   - Education/training requirements

2. **Career Extension Activity**
   - 15-20 minute activity
   - Can be done in-class or as homework
   - Connects episode concept to careers

3. **Career Discussion Questions**
   - "How might a software developer use this concept?"
   - "What other jobs use this type of thinking?"
   - "How does this skill help in different careers?"

#### Example: Episode 1 Teacher Guide Addition

**Career Connections:**
- **Software Engineer:** Uses state management in web applications, mobile apps
- **Game Developer:** Tracks game state (player health, score, level)
- **Systems Analyst:** Models system states and transitions
- **Quality Assurance Tester:** Tests state transitions and edge cases

**Career Extension Activity (15 minutes):**
1. Students choose one career from the list
2. Research: "How does [career] use state management?"
3. Find one real-world example
4. Share findings with class (2-minute presentations)

**Career Discussion Questions:**
- "Can you think of a video game that uses state management?"
- "How might a mobile app developer use states?"
- "What happens if a program loses track of its state?"

### Phase 4: Student Portfolio Development
**Build career-ready portfolio using BallCODE projects**

#### Portfolio Components

1. **Project Showcase**
   - Each episode = one project
   - Screenshot/video of game exercise
   - Code samples (block or Python)
   - Reflection on learning

2. **Skills Documentation**
   - Skills learned in each episode
   - Evidence of mastery
   - Connection to career skills

3. **Career Reflection**
   - Career interests
   - Skills alignment
   - Next steps for learning

#### Portfolio Structure

**Section 1: About Me**
- Introduction
- Coding interests
- Career goals

**Section 2: Projects**
- Episode 1: State Management Project
- Episode 2: Conditional Logic Project
- [Continue for all 12 episodes]

**Section 3: Skills**
- Technical skills learned
- Soft skills developed
- Certifications/badges earned

**Section 4: Career Goals**
- Career interests
- Education plans
- Next learning steps

## Assessment and Evaluation

### Formative Assessment
- Career connection discussions
- Extension activity completion
- Portfolio checkpoints

### Summative Assessment
- Career readiness module completion
- Portfolio submission
- Mock interview performance

### Rubrics

**Career Extension Activity Rubric:**
- **Excellent (4):** Clear connection to career, detailed research, thoughtful reflection
- **Good (3):** Good connection, adequate research, basic reflection
- **Developing (2):** Weak connection, limited research, minimal reflection
- **Needs Improvement (1):** No clear connection, no research, no reflection

**Portfolio Rubric:**
- **Project Quality:** Evidence of learning, code samples, reflections
- **Skills Documentation:** Complete skills list, evidence provided
- **Career Reflection:** Clear goals, realistic plans, next steps identified
