# BallCODE + IBM: Gaming Partnership

**We have a game platform that teaches kids coding through basketball—and the real data that makes it realistic.**

## The Opportunity

**IBM's Challenge:** You need a way to engage kids in STEM education that actually works.

**Our Solution:** BallCODE is a storytelling game that teaches coding, math, and AI through basketball. Kids love it because it's engaging. It works because we use real basketball data (from our BTE Analytics platform) to make scenarios authentic.

**The Leverage:** We have the only sports-to-code platform kids actually use. IBM has the reach. Together, we transform STEM education.

---

## What BallCODE Is

**The Game:** Story-driven learning (70% story, 30% instruction) with interactive exercises that take students from block coding to Python.

**Why It's Realistic:** We use actual basketball statistics and analytics from BTE Analytics to make game scenarios authentic—real player data, actual game situations, authentic context.

**What Makes It Work:** Kids already understand basketball, so learning feels natural. Progressive pathway from sports language → real programming.
