## Gold-Standard PR Exemplars

### Lightweight (Simple)
```
## AIMCODE Summary
- Complexity: [Simple]
- Decision Tree: [Planning]

## CLEAR (brief)
- C - Clarity: Add section headers and fix spacing in Episode 2 outline.
- L - Logic: Update outline → review → publish.
- E - Examples: Episode 1 outline.
- A - Adaptation: Timeboxed to 30 min.
- R - Results: Outline updated; PR approved same day.

## BML
- Build: Outline edits
- Measure: Time-to-merge (<1 day)
- Learn: Keep "##" headers, no <br>
```
