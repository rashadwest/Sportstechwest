## Schedule (v1 — 5 working sessions)

### Session 1 (2–3 hours) — Assets + core pages
- Capture screenshots (6–10): interface overview + Pound/Crossover/In&Out try-it + success + blocks↔python card.
- Decide demo URL + generate QR + short link.
- Draft Pages 1–3, 5–6, 8–9, 11–12 (kid lines + try-it instructions).

### Session 2 (60–90 min) — Story beats
- Write Story beats (Pages 4, 7, 10): 1–3 lines each.

### Session 3 (60–90 min) — Superpowers + upgrades
- Write Pages 13–17 (loops, if/else, bucket types, passing, footwork preview).

### Session 4 (60–90 min) — Bridge + CTA (parent anchors)
- Write Pages 18–20 (Blocks↔Python, Python peek, challenges + CTA).
- Consistency pass: 1–3 lines/page; kids-first tone; parent anchors only where marked.

### Session 5 (2–3 hours) — Layout + publish
- Assemble pages into print-ready PDF + web PDF.
- Apply BallCODE watermark.
- Publish: Gumroad + website checkout.
- Prepare Amazon KDP paperback package (interior + cover).
