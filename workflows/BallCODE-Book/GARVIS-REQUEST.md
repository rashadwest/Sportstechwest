# Garvis Request

**Template for file-based Garvis requests**

---

## ONE Thing

[Clear single objective - what is the ONE thing you want accomplished?]

---

## Tasks

1. [Task 1 - specific, actionable]
2. [Task 2 - specific, actionable]
3. [Task 3 - specific, actionable]

---

## Context

[Optional: Additional context, constraints, or requirements]

---

## Success Criteria

[Optional: How will you measure success? What does "done" look like?]

---

## Usage

1. Fill out this template
2. Save as `GARVIS-REQUEST.md` in project root
3. Run: `python scripts/garvis-command.py --file GARVIS-REQUEST.md`
4. Walk away - Garvis handles everything

---

**Example:**

```markdown
# Garvis Request

**ONE Thing:** Complete Book 2 story

**Tasks:**
1. Write complete story (550-650 words)
2. Update curriculum schema with Book 2
3. Generate website page for Book 2
4. Deploy to Netlify

**Context:** 
- Target audience: Grades 3-8
- Focus on sequences and step-by-step thinking
- Align with CSTA standards

**Success Criteria:**
- Story written and reviewed
- Curriculum updated
- Website live with Book 2 page
- All systems synced
```
