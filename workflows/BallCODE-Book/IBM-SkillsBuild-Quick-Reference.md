# IBM SkillsBuild Integration - Quick Reference
## How to Use Career Readiness Components in BallCODE

**Purpose:** Quick guide for integrating IBM SkillsBuild Teacher Toolkit components into BallCODE curriculum

---

## What is IBM SkillsBuild Teacher Toolkit?

A free resource from IBM that provides:
- **Career Exploration:** Guides for career planning and job market understanding
- **Resume Writing:** Strategies for creating effective resumes
- **Interview Preparation:** Skills for job interviews
- **Flexible Lesson Plans:** Activities adaptable to existing curricula

**Website:** https://skillsbuild.org/secondary-educators/teacher-toolkit

---

## Integration Approach

### Option 1: Add to Existing Episodes (Recommended First Step)
**What:** Add career connections to each of the 12 episodes  
**Time:** 15-20 minutes per episode  
**Effort:** Low - Medium

**How:**
1. Add "Career Connections" section to each teacher guide
2. Include 3-5 careers that use the coding concept
3. Add one extension activity per episode
4. Include discussion questions

**Example:** See `Episode-1-Teacher-Guide.md` - Career Connections section added

**Benefits:**
- Enhances existing curriculum without major changes
- Increases student engagement and motivation
- Shows real-world relevance
- Low implementation effort

---

### Option 2: Create Career Readiness Module (Post-Episode 12)
**What:** Standalone 3-workshop module for career preparation  
**Time:** 6-9 class periods total  
**Effort:** Medium - High

**Workshops:**
1. **Career Exploration** (2-3 periods)
   - Explore tech careers
   - Map coding skills to careers
   - Industry guest speaker/video

2. **Resume Building** (2-3 periods)
   - Use BallCODE projects as experience
   - Create technical resume
   - Build portfolio

3. **Interview Preparation** (2-3 periods)
   - Technical interview basics
   - Mock interviews
   - Practice explaining coding concepts

**Benefits:**
- Comprehensive career preparation
- Portfolio development
- Interview skills
- Career-ready students

---

### Option 3: Portfolio Development (Ongoing)
**What:** Students build portfolio throughout all 12 episodes  
**Time:** 5-10 minutes per episode  
**Effort:** Low

**How:**
- After each episode, students add:
  - Project screenshot/video
  - Code sample
  - Reflection on learning
  - Skills documented

**Benefits:**
- Career-ready portfolio
- Evidence of learning
- Resume material
- Low ongoing effort

## Template for Adding Career Connections

### For Each Episode, Add:

```markdown
## Career Connections
*[Integrated with IBM SkillsBuild Teacher Toolkit]*

### Career Spotlight: [Coding Concept] in Tech Careers

[Brief explanation of how this concept is used in careers]

#### Careers That Use [Concept]

1. **Career Name**
   - **What they do:** [Description]
   - **How they use [concept]:** [Specific application]
   - **Real example:** [Concrete example]
   - **Education:** [Requirements]

[Repeat for 3-5 careers]

#### Career Connection Activity (15-20 minutes)

**Objective:** [Learning goal]

**Instructions:**
1. [Step 1]
2. [Step 2]
3. [Step 3]

**Discussion Questions:**
- [Question 1]
- [Question 2]
- [Question 3]

**Assessment:**
- [Criteria]
```
