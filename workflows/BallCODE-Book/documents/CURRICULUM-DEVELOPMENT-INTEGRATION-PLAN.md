## ðŸ“‹ CURRICULUM DEVELOPMENT WORKFLOW

### Phase 1: Concept Development (AIMCODE Validation)

**Duration:** 1-2 weeks  
**Output:** Validated book proposal

#### Step 1.1: Identify Learning Objective
**AIMCODE Questions:**
- **Zhang:** What basketball context fits this concept?
- **Resnick:** What will students build?
- **Reggio:** What are the multiple entry points?
- **Hassabis:** What does this build on?
- **Jobs:** How do we make this simple?

**Deliverables:**
- Python concept identified
- Basketball context identified
- Grade level determined
- Prerequisites identified

---

#### Step 1.2: Create Book Proposal
**Template:** `BOOK-[N]-PROPOSAL-[CONCEPT].md`

**AIMCODE Validation Checklist:**
- [ ] **Zhang:** Story-first approach (basketball framework)
- [ ] **Resnick:** Constructionist (students build)
- [ ] **Reggio:** Multiple entry points (story, blocks, game, code)
- [ ] **Hassabis:** Systematic progression (builds on previous)
- [ ] **Jobs:** Simple design (intuitive, accessible)

**Required Sections:**
1. Title & Concept
2. Learning Objectives (Measurable)
3. Basketball Story Framework (3-Act Structure)
4. Python Progression (Phase 1, 2, 3)
5. Standards Alignment (CSTA, Common Core, NGSS)
6. Game Integration
7. AIMCODE Expert Alignment

**Deliverable:** Complete, AIMCODE-validated book proposal
