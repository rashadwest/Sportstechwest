## What Is BallCODE?

## How It Works

### Story-First Learning (70% Story, 30% Instruction)

Each lesson is a basketball adventure that naturally teaches coding concepts:

**Example Episode 1: "The Tip-off Trial"**
- Students read a story about a basketball game
- They learn about "state management" (tracking if the ball is in play)
- They practice by fixing mistakes a robot makes
- They learn real coding concepts without realizing it

### Three-Phase Pathway

1. **Sports Language (Block Coding)** - Drag blocks together using basketball terms
2. **Transition Bridge** - See how blocks map to code
3. **Python Learning** - Write real Python code

**End Result:** Kids can actually program in Python.
