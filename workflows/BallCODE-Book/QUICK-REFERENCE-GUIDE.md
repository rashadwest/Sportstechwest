## ðŸŽ“ GRADE-LEVEL QUICK REFERENCE

### Grade 6: Foundation Building
- **Focus:** Introduction to coding concepts and Python basics
- **Books:** 1-3 (Sequences, Conditionals, Loops)
- **Assessment:** 60% blocks, 20% code reading, 20% code writing
- **Outcome:** Can write simple Python code

### Grade 7: Skill Development
- **Focus:** Intermediate Python and data analysis
- **Books:** 1-3 (Review & Extension)
- **Assessment:** 50% Python writing, 30% data analysis, 20% game exercises
- **Outcome:** Can analyze data and create visualizations

### Grade 8: Mastery & Preparation
- **Focus:** Advanced Python and BallCODE Jr. Analytics
- **Books:** 1-3 (Mastery) + BallCODE Jr. Analytics Program
- **Assessment:** 40% Python projects, 30% analytics, 30% portfolio
- **Outcome:** Professional portfolio, ready for high school
