## Instructions for AI

When you receive this filled-out template, you should:

1. **Generate complete blog content** from the Main Content/Notes section
2. **Format properly:**
   - Use `##` for section headers only (never `###`)
   - Separate paragraphs with blank lines (no `<br>` tags)
   - Use `---` on its own line between major sections
   - Wrap images in `<div class="text-center my-4">` blocks
3. **Create front matter:**
   - All required fields
   - Date validated (2025, after latest post)
   - Tags/categories in brackets format: `[tag1, tag2]`
4. **Generate image prompts** if descriptions provided but prompts are blank
5. **Generate social media excerpts** if left blank
6. **Generate SEO description** if left blank
7. **Create Notion database entry** with all fields
8. **Validate formatting** (no `<br>`, correct headers, proper image wrappers)
