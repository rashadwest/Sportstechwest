- **AIMCODE** = Auto-applied for in-depth/comprehensive prompts (ANY domain - system design, research, analysis, workflows, etc.)

- "Research the best..." (system design, workflow, integration, etc.)
- "Analyze comprehensively..." (architecture, methodology, process, etc.)
- "Go in depth on..." (any complex topic)
- "What's the best approach..." (design, implement, solve, etc.)
- "Deep dive into..." (any topic requiring deep analysis)
- "Investigate thoroughly..." (problems, solutions, approaches)
- "Systematic approach..."
- "Expert consultation..."
- "Design a system for..."
- "Create a workflow for..."
- "Analyze the architecture..."

### AIMCODE Ed (Educational) Auto-Applied When You Say:
- "Create a lesson plan..."
- "Develop curriculum..."
- "Write a teacher guide..."
- "Create an assessment..."
- "Design differentiation..."
- "Align to standards..."
- "For Book 1..."
- "For grades 3-5..."

## üîß Override Commands

**Force AIMCODE:**
```
@AIMCODE [your prompt]
```

**Force AIMCODE Ed:**
```
@AIMCODE Ed [your prompt]
```

**Skip Frameworks:**
```
@skip [your prompt]
```
