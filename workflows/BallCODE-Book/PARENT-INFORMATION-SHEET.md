## üéÆ HOW DOES IT WORK?

### Three-Phase Learning Pathway

**Phase 1: Sports Language (Block Coding)**
- Your child learns coding using basketball terms
- Drag blocks together (no typing required)
- See immediate results in game exercises

**Phase 2: Transition Bridge**
- Your child sees how blocks connect to code
- Understand that blocks = real code
- Build confidence for writing code

**Phase 3: Python Learning**
- Your child writes real Python code
- Same concepts, now in professional code
- Creates real programs that work

## üèÄ WHY BASKETBALL?

**Basketball is the framework, Python is what students learn.**

- **70% Story, 30% Skill:** Your child reads engaging basketball stories where coding concepts naturally appear
- **Familiar Context:** Kids already understand basketball, so learning feels natural
- **Real Application:** Students see how coding solves real basketball problems
- **Engaging:** Stories keep students interested while building real skills

**Example:** Instead of "Today we're learning about conditionals," students read a story about a player making decisions during a game, and the coding concept emerges naturally from the story.

## ü§ù HOW CAN I HELP?

### Supporting Your Child

## ‚ùì FREQUENTLY ASED QUESTIONS

### Q: Does my child need prior coding experience?
**A:** No! BallCODE is designed for beginners. We start with visual blocks and progress to Python.

### Q: What if my child doesn't like basketball?
**A:** That's okay! The basketball stories are engaging narratives. Many students enjoy the stories even if they're not basketball fans.

### Q: Is this just for boys?
**A:** Absolutely not! BallCODE is for all students. The stories feature diverse characters, and coding is for everyone.

### Q: Will this help my child in high school?
**A:** Yes! Students who complete BallCODE are prepared for advanced programming courses and have a clear pathway to data science careers.

### Q: What if my child struggles?
**A:** That's normal! Coding takes practice. Teachers provide support, and there are multiple ways to learn (story, game, code, visual).

### Q: Is this aligned with school standards?
**A:** Yes! BallCODE meets all major 6th-8th grade standards (CSTA, Common Core Math, NGSS, ELA).
