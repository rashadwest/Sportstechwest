# Simple Writing Outline - Book 1: The Foundation Block
## Picture Book Format (~30 words per page)

## TITLE
**The Foundation Block**

---

## FORMAT
**Words per page:** ~30 words  
**Total pages:** ~10-15 pages  
**Total words:** ~300-450 words  

---

## PLOT (3 Acts - Broken into ~13 pages)
**What happens:**
- Full-court press defense traps the team
- Nova tries fancy moves (crossovers, spins) - all fail
- Team can't advance past half-court
- Arc detects: "Foundation missing. Basic sequence not established."
- Coach Circuit says: "Start with the foundation. Everything else builds from there."
- Nova realizes: "Maybe I need to start simpler"

**End Act I:** Nova decides to try the basic move

---

### ACT II: The Foundation (~400-500 words)
**What happens:**
- Press gets worse, time running out
- Nova practices the pound dribble (most basic move)
- She sees it as a "block" she can repeat
- Arc: "Pound. Pound. Pound. Repeat." (pattern recognition)
- Math: Counting dribbles (1, 2, 3...), time (each pound = 0.5 seconds)
- Coding: Block 1 (Pound) can be repeated → START → Block 1 → Block 1 → Block 1
- Nova resists at first: "Pound is too simple, I want crossovers"
- Coach: "Master the foundation, then build"
- Nova realizes: "This is like a code block I can repeat!"

**End Act II:** Nova understands foundation blocks

---

### ACT III: Breaking Through (~400-500 words)
**What happens:**
- Final seconds, last chance to break the press
- Nova uses pound dribble sequence: Block 1 repeated
- Team breaks the press! Advances past half-court
- Success using simple, repeated foundation blocks
- Nova: "The foundation works! Now I can build on it."
- Connection: "Coding starts with simple blocks, just like dribbling"

**End Act III:** Team succeeds, Nova understands foundations

---

## CHARACTERS

### NOVA (Main Character)
- Point guard, sees court like code
- Frustrated at first, wants to do fancy moves
- Learns that basics matter
- Transforms: From wanting complex → understanding simple is powerful

### ARC (AI Assistant)
- Detects problems, gives hints
- Key lines:
  - "Foundation missing. Basic sequence not established."
  - "Pattern detected: foundation block missing."
  - "Pound. Pound. Pound. Repeat."

### COACH CIRCUIT
- Wise mentor
- Key lines:
  - "Start with the foundation. Everything else builds from there."
  - "Master the foundation, then build."

### SHADOW PRESS SCOUTS (The Challenge)
- Full-court press defense
- Creates chaos, forces turnovers
- Makes team realize they need basics

## WHAT STUDENTS LEARN

### Basketball
- Pound dribble (most basic dribble)
- Breaking a full-court press
- Foundation moves are important

### AI
- Pattern recognition
- Sequences: "Pound. Pound. Pound. Repeat."

### Math
- Counting: 1, 2, 3, 4...
- Time: Each pound = 0.5 seconds
- Addition: 3 pounds = 1.5 seconds

### Coding
- Basic blocks (Block 1 = Pound)
- Sequences: Repeat blocks to create movement
- Foundation first: Master basics before complex
