## ðŸ’¡ Key Concepts

### Categories
- **Insights** - Personal experiences, thoughts, learnings
- **Tutorial** - Step-by-step guides, how-tos
- **News** - Updates, announcements, events

### Tags
Use 2-7 relevant tags. Common: `automation`, `AI`, `server`, `docker`, `mcp`, `productivity`

### Content
Provide an outline, notes, or draft. AI expands it into a complete post.

### Images
Describe what images should show. AI generates prompts and places them correctly.

### Formatting
AI ensures:
- No `<br>` tags
- Headers use `##` only
- Images wrapped properly
- Valid front matter
- Date validation
