## üêç PYTHON GAME MODE OVERVIEW

### How It Works

**Same as Block Coding, But with Python:**

1. **Students write Python code:**
   ```python
   dribble('move', 0.5)
   dribble('pound', 0.5)
   shoot()
   ```

2. **Code executes on basketball court:**
   - Same visual feedback as block coding
   - Same basketball context
   - Same challenges and terminology

3. **Progressive difficulty:**
   - Level 1: Simple sequences
   - Level 2: Variables
   - Level 3: Conditionals
   - Level 4: Loops
   - Level 5: Functions

### Basketball Python API

**Core Functions:**
- `dribble(action, clock)` - Execute dribble move
- `pass_to(direction)` - Pass the ball
- `shoot()` - Shoot the ball
- `move(direction, distance)` - Move player
- `if_defender(condition, action)` - Conditional logic
- `repeat(times, action)` - Loop structures

### Block-to-Python Bridge

**Side-by-Side View:**
- Blocks on left, Python on right
- Click block ‚Üí see Python equivalent
- Click Python ‚Üí highlight corresponding block
- Gradual transition support
