# Cursor Chat Sidebar Fix Guide

**Issue:** Chat history sidebar not visible on the left side of Cursor

---

## üîß QUICK FIXES TO TRY

### Method 1: Keyboard Shortcut (Fastest)
1. Press **`Cmd + B`** (Mac) or **`Ctrl + B`** (Windows/Linux)
   - This toggles the sidebar visibility
   - Try pressing it once or twice to restore the sidebar

### Method 2: View Menu
1. Click **View** in the top menu bar
2. Look for **"Appearance"** or **"Sidebar"** options
3. Select **"Show Sidebar"** or **"Toggle Sidebar"**

### Method 3: Command Palette
1. Press **`Cmd + Shift + P`** (Mac) or **`Ctrl + Shift + P`** (Windows/Linux)
2. Type: **"Toggle Sidebar"** or **"View: Toggle Sidebar"**
3. Press Enter

### Method 4: Settings
1. Press **`Cmd + ,`** (Mac) or **`Ctrl + ,`** (Windows/Linux) to open Settings
2. Search for **"sidebar"**
3. Look for sidebar visibility settings
4. Enable sidebar if it's disabled

### Method 5: Reset Layout
1. Press **`Cmd + Shift + P`** (Mac) or **`Ctrl + Shift + P`** (Windows/Linux)
2. Type: **"Reset Layout"** or **"View: Reset Layout"**
3. This will restore default window layout including sidebars

---

## üéØ SPECIFIC TO CHAT SIDEBAR

If the main sidebar is visible but chat history is missing:

1. **Check Chat Panel:**
   - Look for a chat icon in the sidebar
   - Click it to open the chat panel
   - The chat history should appear within the chat panel

2. **Open New Chat:**
   - Press **`Cmd + L`** (Mac) or **`Ctrl + L`** (Windows/Linux) to open chat
   - Chat history should appear in the left panel of the chat interface

3. **Check Chat Settings:**
   - In the chat interface, look for a settings/gear icon
   - Check if there's an option to show/hide chat history

---

## üîç TROUBLESHOOTING

### If Sidebar is Collapsed (Very Thin)
- Look for a thin vertical bar on the left edge
- Hover over it - it should expand
- Or click the left edge to expand

### If Sidebar is Off-Screen
- Try resizing the window
- Check if you have multiple monitors - sidebar might be on another screen
- Try moving the window or resetting window position

### If Nothing Works
1. **Restart Cursor:**
   - Quit Cursor completely
   - Reopen it
   - Sidebar should restore to default state

2. **Reset Workspace:**
   - Close all windows
   - Reopen your workspace
   - Sidebar should appear

3. **Check Cursor Updates:**
   - Make sure you're on the latest version
   - Updates sometimes fix UI issues
