# BallCODE Logo Assets

## Logo File Location

**Place your BallCODE logo here:**
- `ballcode-logo.png` - Main logo (recommended: 120x60px or 240x120px)
- `ballcode-logo.svg` - Vector version (scalable)

---

## Logo Description (For Reference)

**BallCODE Logo Design:**

**Left Side - Stylized Basketball Graphic:**
- Semi-circular shape (left half of basketball) with dark blue outline (#1a3d5c)
- Basketball seams: horizontal line bisecting, curved vertical line
- Top-right segment: Vibrant orange (#ff6b35)
- Bottom-right segment: Dark blue (#1a3d5c)
- Left half: Light blue (#4a90e2) with gradient/3D effect

**Right Side - Text and Code Symbols:**
- "BALL" stacked above "CODE" (all caps, bold, dark blue #1a3d5c)
- Above "BALL": Symbols `=> >> - •` (dark blue)
- Below "CODE": Symbols `> = > - •` (dark blue)
- Command-line/programming syntax aesthetic

**Color Palette:**
- Primary Dark Blue: `#1a3d5c`
- Light Blue: `#4a90e2`
- Orange: `#ff6b35`
- Background: White

---

## How to Add Logo

1. **Get logo file:**
   - Export logo as PNG (120x60px or 240x120px recommended)
   - Or use SVG for scalable version

2. **Save to this directory:**
   - Name: `ballcode-logo.png` or `ballcode-logo.svg`
   - Location: `email_system/assets/images/`

3. **Signature will automatically use it:**
   - HTML signature references: `assets/images/ballcode-logo.png`
   - Update signature HTML if using different filename

---

## Logo Usage in Email Signature

The signature HTML references the logo as:
```html
<img src="assets/images/ballcode-logo.png" alt="BallCODE Logo" width="120" height="60">
```

**Note:** For email signatures, you may need to:
- Host logo online (e.g., on ballcode.co) and use full URL
- Or embed logo as base64 (for offline use)
- Or attach logo to each email (not recommended)

---

## Recommended Logo Sizes

- **Email Signature:** 120x60px (small, professional)
- **Email Header:** 240x120px (larger, more prominent)
- **SVG:** Any size (scalable, best quality)
