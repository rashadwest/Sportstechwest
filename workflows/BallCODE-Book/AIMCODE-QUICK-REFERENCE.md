## Story Writing Checklist

### Act I: Setup (Zhang)
- [ ] Starts with basketball action
- [ ] Basketball problem/challenge introduced
- [ ] Basketball context is clear

### Act II: Struggle (Hassabis + Resnick)
- [ ] AI/math/coding concept emerges from basketball need
- [ ] Students see concept solving basketball problem
- [ ] Hands-on building activity included

### Act III: Resolution (Zhang + Jobs)
- [ ] Basketball success demonstrates learning
- [ ] Solution is simple and clear
- [ ] "It just works" - natural resolution

### Integration (Reggio)
- [ ] Story connects to game mode
- [ ] Code exercises connect to story
- [ ] Visuals support all modes
- [ ] Multiple entry points available
