## Key Concepts

### Three Literacies Defined

1. **AI Literacy:** Understanding, interacting with, and critically evaluating AI systems
2. **Math Literacy:** Applying mathematical concepts to solve real-world problems
3. **Coding Literacy:** Understanding and writing code, applying computational thinking

### Integration Principle

**Basketball is the framework, but AI + Math + Coding are what students learn.**

- Basketball = Engaging narrative vehicle
- AI + Math + Coding = Actual learning objectives
- All three taught together, not separately

## Quick Reference: Episode 1 Example

### AI Literacy (Grades 3-5)
- AI can "see" and detect changes

## How to Use This System

### For Curriculum Developers

1. **Start with Strategy Document**
   - Read `Literacy-Integration-Strategy.md`
   - Understand the framework
   - Review learning objectives

2. **Reference Research Sources**
   - Check `Research-Sources/` for citations
   - Understand research backing
   - Cite sources in materials

3. **Apply to Episodes**
   - Define literacy objectives for each episode
   - Integrate all three literacies
   - Document evidence in episode

4. **Create Teacher Resources**
   - Include literacy objectives in teacher guides
   - Provide assessment rubrics
   - Reference research sources

### For Teachers

1. **Review Literacy Objectives**
   - Read the "Literacy Learning Objectives" section
   - Understand what students will learn
   - Note grade-level differences (3-5 vs 6-8)

2. **Plan Instruction**
   - Identify where each literacy appears in story
   - Prepare to explicitly teach each literacy
   - Plan assessment for each literacy

3. **Assess Learning**
   - Use rubrics to assess each literacy
   - Check evidence of learning
   - Provide feedback on all three literacies

4. **Reference Research**
   - Understand "why" behind approach
   - Share with parents/administrators
   - Use in professional development
