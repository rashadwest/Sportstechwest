## WRITING TIPS

### DO:
- ‚úÖ Start each scene with action
- ‚úÖ Use basketball terminology naturally
- ‚úÖ Show, don't tell (show frustration, don't just say "Ava was frustrated")
- ‚úÖ Let concepts emerge from basketball needs
- ‚úÖ Use dialogue to reveal character and concepts
- ‚úÖ Keep it engaging and adventurous

### DON'T:
- ‚ùå Don't explain concepts before basketball context
- ‚ùå Don't start with "Today we're learning about..."
- ‚ùå Don't make it feel like a lesson
- ‚ùå Don't skip the basketball action
- ‚ùå Don't make concepts feel forced
