## üìã THE FRAMEWORK: S-M-A-R-T-PED

### **S - Specify the Role**
Define AI's educational persona
```
Example: "Act as an experienced K-5 CS educator teaching Python through basketball stories"
```

### **M - Mention the Purpose**
State the educational goal
```
Example: "Create a lesson plan for Book 1, focusing on sequences"
```

### **A - Define the Audience**
Identify learners
```
Example: "Grades 3-5 (ages 8-11), mixed ability levels"
```

### **R - Pedagogical Constraints**
Set educational parameters

**Standard:**
- Bloom's Taxonomy
- Standards (CSTA/Common Core/NGSS)
- Course Outcomes
- Real-world examples

**BallCODE Enhancements:**
- Multiple entry points (Reggio)
- Alpha Evolve layers (Hassabis)
- Basketball framework (Zhang)
- Constructionist activities (Resnick)
- Simplicity (Jobs)
- Differentiation

### **T - Output Format**
Specify structure needed
```
Example: "Lesson plan with: Hook, Instruction, Practice, Closure, Assessment"
```

### **PED - Prompt ‚Üí Evaluate ‚Üí Develop**
Iterative refinement cycle
