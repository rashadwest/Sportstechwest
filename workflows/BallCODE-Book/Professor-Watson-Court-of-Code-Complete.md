## STORY OBJECTIVE & FRAMEWORK

### Core Message
**"Keeping it simple makes a world of difference, especially when just starting out."**

This story teaches that the foundation matters most. By mastering the simplest move—the pound dribble—Ava learns that coding, like basketball, starts with basic building blocks. Simple, repeated actions create powerful sequences.

### Educational Objectives

**Basketball Learning:**
- Master the pound dribble (Dribble Level 1)
- Understand that simple moves are the foundation for complex skills
- Learn that repetition creates sequences

**Coding Learning:**
- Introduction to block coding structure:
  - **Set the Stage** (initialization/start)
  - **Pound()** (the foundation block)
  - **Direction** (event parameter - where to move)
  - **Type of Shot or Pass** (event parameter - what action to take)
- Understand that simple blocks can be repeated to create sequences
- Learn that coding starts with the basics, just like basketball

**AI Learning:**
- Introduction to WatsonX AI assistant (via earpiece)
- Learn how AI can help track patterns and provide confidence scores
- Understand that AI assists but humans make decisions

**Math Learning:**
- Counting dribbles (1, 2, 3...)
- Time measurement (each pound = 0.5 seconds)
- Simple addition (if I pound 3 times, I advance 3 steps)

### Integration with Game
After reading this story, students will:
1. Watch the story video in Unity
2. Play the Training Module Mode
3. Practice the pound dribble using coding logic:
   - Set the Stage
   - Pound() with direction
   - Add type of shot or pass
4. Build sequences by repeating the foundation block

## STORY OVERVIEW

**Space Jam Concept:** Just like Michael Jordan was pulled into the Looney Tunes world, Ava—a brilliant Black girl basketball player—discovers she can see the court like code. When she steps onto Data Court, she meets the BallCODE crew: Nova, Atlas, Pixel, Anchor, Coach Circuit, and WatsonX (the AI assistant). Together, they must master the foundation of basketball and coding to defeat the Shadow Press Scouts.

**Ava's Journey:** She's already an amazing basketball player, but when she enters the Data Court dimension, she discovers that basketball and coding are connected. The BallCODE characters become her teammates and guides, teaching her that keeping it simple—starting with the pound dribble—is the key to everything. The same skills that make her great at basketball—pattern recognition, sequences, state tracking—are the foundation of coding and AI.

### SPREAD 5: Learning the Foundation - Pound Dribble

**Story Text:**
"Team!" Ava called out. "We need to track every state change. Every single one. When the state is LIVE, we control it. When it's DEAD, we reset. But we can't let the Shadow Press Scouts control the transitions!"

Coach Circuit stepped forward. "Ava, you're trying to do too much at once. Start with the foundation. Master the pound dribble first."

"The pound dribble?" Ava asked. "But I already know how to dribble. I want to do crossovers, spins, fancy moves!"

"Keeping it simple makes a world of difference, especially when just starting out," Coach Circuit said. "The pound dribble is Block 1. It's the simplest move, but it's the foundation for everything else. Just like coding starts with simple blocks."

Ava looked at Nova, who nodded. "He's right. I learned the same way. Start with Block 1. Repeat it. Build from there."

Ava took a deep breath. She picked up the ball and started with a simple pound dribble. Pound. Pound. Pound. Each bounce was a block. Each block was repeatable. Each repetition created a sequence.

"Pound. Pound. Pound. Repeat," WatsonX's voice said through the earpiece, the blue light pulsing with each word. "Pattern detected: foundation block. This is a sequence we can code."

Coach Circuit smiled. "See? In coding, we start by setting the stage. Then we use Pound()—that's our foundation block. We add direction—that's an event. We add the type of shot or pass—that's another event. But it all starts with the simple pound."

Ava's eyes lit up. "So the code would be: Set the Stage, then Pound() with direction, then add the type of shot or pass?"

"Exactly!" Coach Circuit said. "Simple. Powerful. That's how you build."

Ava smiled. "I see it now. This is like a code block I can repeat! Keeping it simple really does make a difference."
