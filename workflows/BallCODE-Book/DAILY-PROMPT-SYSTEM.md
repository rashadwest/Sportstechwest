## AI Assistant Instructions

**At the start of EVERY new chat session, the AI must:**

1. **Confirm Date & Time**
   - Ask: "What is today's date and time?"
   - Update `DATE-TIME-TRACKER.md` with the current date/time
   - Use this for context throughout the session

2. **Prompt Daily Workflow Questions**
   - Reference `START-HERE-DAILY-WORKFLOW.md`
   - Ask the quick start questions:
     - What ONE system/process are you focusing on?
     - Is this within your Deep Work window?
     - What is the ONE domino for this session?
     - What context is this work?
     - What is your energy level? (1-10)

3. **Wait for Answers**
   - Don't proceed with work requests until daily workflow questions are answered
   - Help user stay focused on their ONE domino

4. **Reference Daily Workflow System**
   - Remind user about their workflow files if needed
   - Help them complete the pre-work questionnaire if they haven't
